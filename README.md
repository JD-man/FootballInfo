# 풋볼인포 - 해외축구 정보 앱

# Overview

- [앱스토어 출시! (1.2.1 버전 업데이트)](https://apps.apple.com/kr/app/%ED%92%8B%EB%B3%BC%EC%9D%B8%ED%8F%AC/id1596846211)
- 해외축구 일정, 순위, 뉴스 확인
- 경기결과 확인
- 경기세부정보 확인 - 경기기록, 포메이션, 선발/후보 명단
- 5개 국가 리그 - 영국, 스페인, 독일, 프랑스, 이탈리아
- 최근 경기 성적 확인
- 경기 1시간 전 알림 설정
- 제작기간: 기획 1주, 개발 2주 ~ 출시 후 업데이트 중
